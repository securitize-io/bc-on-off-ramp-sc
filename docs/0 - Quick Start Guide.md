# Securitize On/Off Ramp Protocol - Quick Start

Welcome to the Securitize On/Off Ramp Protocol documentation. This guide will walk you through the complete deployment and testing process.

## Overview

The Securitize On/Off Ramp Protocol enables:

- **On-Ramp**: Purchase of digital securities with liquidity tokens
- **Off-Ramp**: Redemption of digital securities for liquidity tokens (e.g., USDC)

This documentation focuses on the Off-Ramp deployment and testing process.
